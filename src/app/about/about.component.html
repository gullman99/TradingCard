About

Author: Michael Gulson
Date: 8/1/2021

Trading Card Inventory allows the user to add, delete, and update trading cards. The total estimated value is calculated and displayed in the top right corner.
Trading Card Inventory uses Angular CLI and Ngrx framework including reducers to keep track of state.
It includes an API call to get an image for sports players. Try entering a famous sports player like Lebron James!
